# Delete API Key

Delete a specific API key

DELETE

/

v1

/

/

/

Try it

Delete API key

cURL

Copy

```
curl --request DELETE \
  --url https://api.requesty.ai/v1/manage/apikey/{api_key_id} \
  --header 'Authorization: Bearer <token>'
```
