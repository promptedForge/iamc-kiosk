# OpenWebUI

Requesty routing for OpenWebUI

You can easily integrate Requesty with OpenWebUI to access all available models through a single interface.

## [â€‹](https://docs.requesty.ai/applications/openwebui\#setup-instructions)  Setup Instructions

1. Go to the Admin Settings in OpenWebUI
2. Change the OpenAI URL to the Requesty router URL: `https://router.requesty.ai/v1`
3. Add your API key, which you can create on the [API Keys Page](https://app.requesty.ai/api-keys) in the platform
4. Save your settings and enjoy access to 150+ models through the familiar OpenWebUI interface

This integration allows you to use any model available in the Requesty platform directly from OpenWebUI, giving you more flexibility and options for your AI interactions.
